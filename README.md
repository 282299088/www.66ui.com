# www.66ui.com

#### 介绍
"小镇故事" 官网微信小程序 织梦CMS小程序

一个网络运营人员，非专业代码编写者  只是在业绩压力比较大的时候 做的小玩意

本小程序 基于"守望轩"开源版本  从新调整接口（织梦CMS程序）已经添加功能 不改动原有代码 简单配置就可以使用。


代码小白 开源出来希望大家一起完善。

主要功能，
首页 
分类
列表
文章
个人中心
个人主页
积分中心  等级
